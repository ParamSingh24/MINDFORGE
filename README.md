
# Param Singh Portfolio

A modern, responsive portfolio website built with React and TypeScript.

## About

This is a personal portfolio website showcasing my skills, projects, and experience as a full-stack developer.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Shadcn UI Components

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

## Building for Production

```bash
npm run build
```
